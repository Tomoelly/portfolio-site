# Elly Huang Portfolio

Personal portfolio site for job applications and case studies.

Live site:

- https://aibuilderfolio.web.app

Source and backup hosting:

- GitHub repo: https://github.com/Tomoelly/portfolio-site
- GitHub Pages backup: https://tomoelly.github.io/portfolio-site/

Includes:

- Portfolio landing page
- LINE Bot backend case study
- Essay contest management system case study
- Resume PDF

## Local Preview

```bash
python3 -m http.server 8765
```

Then open:

```text
http://localhost:8765
```

## Deploy

```bash
firebase deploy --only hosting:aibuilderfolio
```
